# HOW TO USE
- Install ruby.
- Download the puzzle input to a file namned `input` (https://adventofcode.com/2021/day/4/input).
- For puzzle 1, run: `ruby puzzle1.rb input`
- For puzzle 2, run: `ruby puzzle2.rb input`
